# CmsShopp
System do zarządzania treścią oraz funkcjonalność sklepu internetowego

Użyte technologie: 
– ASP.NET MVC
– Entity Framework
– LocalDB
– Rejestracja, logowanie, role
– Widoki, widoki częściowe
– Ajax
– Routes
– Bootstrap
– AdminLTE
– GIT
– CKEditor, Roxy File Manager, Drag And Drop – i inne …
